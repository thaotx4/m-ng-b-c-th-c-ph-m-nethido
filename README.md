# Tên Dự Án (Nethido Landing Page)

Đây là trang Landing Page được thiết kế đơn giản dưới dạng single HTML file.

## 🚀 Tính năng
* Giao diện tương thích (Responsive)
* Tối ưu hóa để deploy nhanh trên Vercel.

## 🛠 Cách sử dụng
1. Tải file `nethido_landingpage_singlefile_vercel.html` về máy.
2. Mở trực tiếp bằng trình duyệt hoặc dùng Live Server trong VS Code.
